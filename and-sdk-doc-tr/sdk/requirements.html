System Requirements——系统要求

The sections below describe the system and software requirements for developing Android applications using the Android Development Tools, revision 3.
下面的章节介绍了使用第三版Android开发工具开发Android程序的系统和软件要求。

Supported Operating Systems——支持的操作系统

  • Windows XP (32-bit) or Vista (32- or 64-bit)——Windows XP (32位) 或 Vista (32位或64位)
  • Mac OS X 10.4.8 or later (x86 only)——Mac OS X 10.4.8 或更高 (仅支持x86平台)
  • Linux (tested on Linux Ubuntu Hardy Heron)——Linux(Linux Ubuntu Hardy Heron上测试通过)
    • 64-bit distributions must be capable of running 32-bit applications. For information about how to add support for 32-bit applications, see the Ubuntu Linux installation notes.     如果您使用64位版本的，则请确定它支持32位的程序，关于如何添加32位程序的支持，请参见Ubuntu Linux installation notes。

Supported Development Environments——支持的开发环境

  • Eclipse IDE——Eclipse集成开发环境
    • Eclipse 3.4 (Ganymede) or 3.5 (Galileo)
      • Note: Eclipse 3.3 has not been tested with the latest version of ADT and support can no longer be guaranteed. We suggest you upgrade to Eclipse 3.4 or 3.5.     注意：我们没有使用Eclipse3.3测试最新版本的ADT插件，不保证可用。我们建议您升级到3.4或3.5版。
      • Recommended Eclipse IDE packages: Eclipse IDE for Java EE Developers, Eclipse IDE for Java Developers, Eclipse for RCP/Plug-in Developers, or Eclipse Classic (3.5.1+)     推荐的Eclipse开发环境包：Eclipse IDE for Java EE Developers, Eclipse IDE for Java Developers, Eclipse for RCP/Plug-in Developers, or Eclipse Classic (3.5.1+) 
      • Eclipse JDT plugin (included in most Eclipse IDE packages)       Eclipse JDT 插件（多数Eclipse包中已包含）
    • JDK 5 or JDK 6 (JRE alone is not sufficient)——JDK 5 or JDK 6 (只有JRE是不够的)
    • Android Development Tools plugin (optional)——Android Development Tools plugin (可选)
    • Not compatible with Gnu Compiler for Java (gcj)——不支持Gnu Compiler for Java (gcj)
  • Other development environments or IDEs——其他开发环境
    • JDK 5 or JDK 6 (JRE alone is not sufficient)——JDK 5 or JDK 6 (仅有JRE是不够的)
    • Apache Ant 1.6.5 or later for Linux and Mac, 1.7 or later for Windows——Linux和Mac上1.6.5及以上，或Windows上1.7及以上的   Apache Ant
    • Not compatible with Gnu Compiler for Java (gcj)——不支持Gnu Compiler for Java (gcj)

Note: If JDK is already installed on your development computer, please take a moment to make sure that it meets the version requirements listed above. In particular, note that some Linux distributions may include JDK 1.4 or Gnu Compiler for Java, both of which are not supported for Android development.
1ot注意：2如果在您的开发用计算机上已经安装了JDK，则请确认版本符合上面的要求。 特别的，有些Linux发行版可能已经带了JDK 1.4或GCJ，这两者都是不被支持的。

Hardware requirements——硬件要求

  • For the base SDK package, at least 600MB of available disk space. For each platform downloaded into the SDK, an additional 100MB is needed.     基本的SDK包需要至少600MB的可用磁盘空间。每个平台组件都需要约100MB的额外空间。
